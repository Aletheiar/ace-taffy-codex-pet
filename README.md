# 永雏小菲 Codex 任务宠物

一个供 Codex Desktop 使用的非官方同人任务宠物，包含待机、移动、挥手、跳跃、任务状态以及 16 方向鼠标观察动画。

![动画预览](docs/contact-sheet.png)

第 8 行“检查任务结果”现在包含小菲版经典唐笑，用于任务完成后提示用户检查和审批结果：

![唐笑动画](docs/tang-smile-review.gif)

第 5 行“任务失败”现在包含小菲版唐哭，用宽蓝泪流、擦眼和蹲下爆哭表现失败后的夸张沮丧：

![唐哭动画](docs/tang-cry-failed.gif)

## 安装

将 `pet/ace-taffy-xiaofei` 复制到 `%USERPROFILE%/.codex/pets/`，完全重启 Codex，然后在设置中的外观或头像选项选择“永雏小菲”。

## 内容

- `pet/ace-taffy-xiaofei/`：可安装宠物包
- `skills/build-codex-task-pet/`：从角色参考图制作 Codex 任务宠物的复用 skill
- `docs/`：预览与验证结果

## 后续优化路线

- 使用 2×或 4×母版生成，再缩小到每格 192×208，提升线条、眼睛和发饰稳定性。
- 重绘 135°、157.5°、202.5°、225°、315°、337.5°等表现较弱的斜向视线。
- 统一各动画行的脸型、发色、蝴蝶结、服装纹样和身体比例；优先放大第 9～10 行观察方向，并校正执行任务行与失败动画中段的头身比例。
- 调整移动与跳跃帧的脚底接触点、重心轨迹和循环首尾衔接。
- 在浅色、深色和高饱和桌面背景上检查透明边缘与缩放锐度。
- 保留无损 PNG 母版，仅将最终运行图集导出为 RGBA WebP。
- 继续微调已加入第 8 行“检查任务结果”的小菲版唐笑节奏，并评估是否增加低概率待机彩蛋；当前正式版本采用“检查结果 → 唐笑 → 恢复满意表情”的六帧序列。
- 继续微调已加入第 5 行“任务失败”的小菲版唐哭节奏；当前正式版本采用“担忧 → 宽蓝泪流 → 擦眼 → 蹲下爆哭 → 恢复难过”的八帧序列。

当前比例审计及建议修复顺序见 [`docs/proportion-audit.md`](docs/proportion-audit.md)。
